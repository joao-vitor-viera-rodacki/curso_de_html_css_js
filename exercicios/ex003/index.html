Testando como fazer as coisas

A tag <sup> :

Esta tag eleva os caracteres que são selecionados

A tag <sub> :

Esta tag abaixa os caracteres que são selecionados

A tag <em> :

Está é semantica que deicha em italico os caracteres selecionados

A tag <strong> :

Esta tag é semantica que deicha em negrito os caracteres selecionados

A tag <mark> :

Esta tag modifica o background-color dos caracteres selecionados

A tag <del> :

Risca os caracteres selecionados

A tag <small> :

Esta tag deicha os caracteres selecionados pequenos

A tag <ins> :

Esta tag sublinha o texto selecionado

A tag <u> :

Esta tag só tem forma não é semantica
